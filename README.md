# Deltasound

Client-side Fabric mod for Hypixel SkyBlock that plays custom sounds from chat triggers (and later dungeon room entry).

## Requirements

- Minecraft **26.1**
- Java **25**
- Fabric Loader **0.19.3+**
- Fabric API

## Development

```bash
./gradlew runClient
./gradlew build
```

See [docs/PORTING.md](docs/PORTING.md) for how we keep Minecraft touch-points thin so ports stay small.
