Deltasound

Client-side sound triggers for chat events. Configure detection strings in-game, import your own .ogg files, and keep the Minecraft touch-points thin for easy version ports.

Download latest View on GitHub Setup

What it does

  • Watches chat for configured text (substring or regex)
  • Plays vanilla or custom sounds when a line matches
  • In-game GUI via /deltasound or /ds
  • Import .ogg sounds (file picker or drag-and-drop onto the GUI)

Setup

  1. Install Fabric Loader for Minecraft 26.1.2 and Fabric API
  2. Drop the Deltasound jar into your mods folder
  3. Join a world and run /ds
  4. Add a detection string (for example RNG Drop!), pick a sound, Save
./gradlew build
# jar → build/libs/deltasound-<version>.jar

Porting

Chat matching lives in pure Java (dev.deltasound.core). Minecraft/Fabric adapters sit in client.bridge so version bumps stay localized. See docs/PORTING.md in the repo.
